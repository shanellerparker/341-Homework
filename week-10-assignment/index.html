Responsive Web Design

Responsive web design. Also known as R.W.D is a type of style for web design that takes into consideration the numerous different devices.This allows for a better viewing expeirence for various screen sizes that people use to view webpages. This approach to web development focuses on the design esthetic for pages that can shift and adjust their layout depending on the screen fit that would best suit a user's experience. R.W.D also benefits the developer by saving work hours. This is because R.W.D is capable of changing a site to the optimal viewing size depending on the device without having to develop multiple site for each.

Additional Information on R.W.D

It's vital for developers to make use of R.W.D in the web pages they develope because of the advancement and wide use of technology. The variety of the devices that have the ability to browes the web also come with a variety of screen sizes. Because of this it is in the best interest of the user and the developer to create web pages where the content/diaglogue are dislplayed to the user in a way that is intentional so the message trying to be conveyed is accurate.

Other Information for R.W.D

Here are a examples of the different elements that used to enhance the responsiveness of a web page:

Flexboxes are CSS elements that allows your webpage to easily adapt to changes in screen size while keeping your content displayed the way it was intended.

Wireframes are basically rough drafts or sketches that used to create an out of what a webpage will look like on different devices and different screen sizes.
